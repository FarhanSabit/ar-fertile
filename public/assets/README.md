# AR Fertilizer Ltd - Assets

This folder is designated for storing all static assets for the AR Fertilizer Ltd project, including:

- **images/**: Product photography, hero backgrounds, and gallery items.
- **icons/**: Brand icons and UI elements.
- **logos/**: Corporate identity files.

All static assets should be organized here to maintain a clean root directory. For the current application state, images provided by the environment are referenced from the project root but can be migrated here for production organization.
